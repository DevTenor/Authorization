## Authenticaction service 🌱

Service for client authorization and registration, written in Spring Boot.

> [!TIP]
> You can run enitre application on local machine by address http://localhost:5173. Enough to execute docker compose file.

<img width="1152" height="648" alt="auth" src="https://github.com/user-attachments/assets/47788495-05e5-4080-a0e7-51e4d3f65128" />


## Features 🛡️

- User registration
- PostgreSQL database support
- Password encryption
- JWT token access
- User authorization
- Editing personal data
- Account removal
- Log out

## Usage 🚣

## Via docker compose

```
docker-compose up -d --build
```
<b>To stop the container</b>

```
docker-compose down -v
```
